# Moducard_System_Handbook
Repo with source files for Moducard System Handbook

# How to modify and generate PDF
### On Windows, using Visual Studio Code
1. Clone this repo
2. Open ./LATEX as folder in VSC
3. Install extension "LaTeX Workshop" by James Yu in VSC
5. We well use MiTeX for LaTeX compiler, as it downloads all the packages as you need them, but you can follow guides from [LateX Workshop wiki](https://github.com/James-Yu/LaTeX-Workshop/wiki/Install#installation) to use other
6. Install [MiTeX](https://miktex.org)
7. Install [Pearl](https://strawberryperl.com)
8. Done

Now you can use<br>
"ctr + s" to save and compile the LaTeX, <br>
"ctr + left mouse button" on PDF in VSC to jump to main.tex code fragment<br>
"ctrl + alt + J" on main.tex to jump to PDF<br>
